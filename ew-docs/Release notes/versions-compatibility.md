| Android SDK | iOS SDK    | Javascript SDK | [Fireblocks js SDK](https://www.npmjs.com/package/fireblocks-sdk) |
| :---------- | :--------- | :------------- | :---------------------------------------------------------------- |
| SDK \< 2.1  | SDK \< 2.1 | SDK \< 10.0    | SDK \< 9.13                                                       |
| 2.1 \<= SDK | 2.1 \<=SDK | 10.0 \<=SDK    | 9.13 \<= SDK                                                      |
